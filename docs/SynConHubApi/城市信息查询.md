# 城市信息查询

根据城市名称的关键字模糊查询平台维护的城市信息

## 1. 地址和方法

* **路径**: "/service/synconhub/common/city/search"
* **方法**：POST

## 2. 请求参数

| 字段     |  类型   | 是否必填 |      说明      |                 备注                  |
| :------- | :-----: | :------: | :------------: | :-----------------------------------: |
| keywords | String  |   必填   | 城市名称关键字 |                                       |
| page     | Integer |    否    |      页数      |        **默认**1，**最小值**1         |
| size     | Integer |    否    |      项数      | **默认**30，**最小值**1，**最大值**30 |

## 3. 请求样例

```java
{
    "keywords": "shenzhen",
    "page": 1,
    "size": 30
}
```

## 4. 响应参数

| 字段名  | 含义                                            |
| ------- | ----------------------------------------------- |
| code    | 状态码                                          |
| message | 提示信息                                        |
| data    | 符合查询条件的所有城市信息,城市信息按照名称排序 |
